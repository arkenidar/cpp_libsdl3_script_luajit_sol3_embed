# Development Plans

This directory contains implementation plans for features being developed in the project.

## Active Plans

| Plan | Description | Status |
|------|-------------|--------|
| [text-widget-plan.md](text-widget-plan.md) | Text rendering and text widget implementation | Phase 3 In Progress |

## Using Plans with Claude Code

These plans are designed to work with [Claude Code](https://claude.com/claude-code). To resume work on a plan:

```
resume docs/plan/text-widget-plan.md
```

Or ask Claude Code to continue with a specific phase:

```
Continue with Phase 2 of docs/plan/text-widget-plan.md
```

## Plan Structure

Each plan follows this structure:

1. **Overview** - High-level description of the feature
2. **Status** - Current progress summary
3. **Phases** - Incremental implementation steps
4. **Implementation Order** - Detailed task checklist
5. **Notes** - Additional context and considerations

## Contributing

When adding new plans:

1. Create a new `.md` file with a descriptive name
2. Follow the existing plan structure
3. Update this README with the new plan
4. Use checkboxes or status indicators for progress tracking
